  Значения параметров запуска имеют приоритет над значениями из файла настроек.
  Параметры указываются по отдельности, разделяются пробелами
  (то же относится и к значениям параметров).
  В случае задания взаимоисключающих параметров или их дублирования
  будет учтён последний указанный параметр/последнее значение.

  Параметры:

  [-a ЗНАЧЕНИЕ|--hw-accel ЗНАЧЕНИЕ]	Включает или отключает использование
					аппаратного ускорения для отрисовки.
					ЗНАЧЕНИЕ вида "1", "y", "Y" или "on"
					(без кавычек) включает данную
					возможность, ЗНАЧЕНИЕ вида "0", "n", "N"
					или "off" (без кавычек) - выключает.
					Данный параметр конфликтует
					с параметром '--sdl12' и отключает его
					(если указан позднее).

  [-b|--show-building]			Включает визуальный показ создания
  					скелета лабиринта.

  [-c ЗНАЧЕНИЕ|--conf ЗНАЧЕНИЕ]		Позволяет использовать файл настройки,
					отличный от файла по умолчанию.
					Если файл, указанный в ЗНАЧЕНИЕ,
					не существует или недоступен
					для чтения и записи пользователем,
					запускающего программу - будет
					использован файл настройки по умолчанию.

  [-d|--debug]				Отображение в терминале схематичного
					изображения лабиринта и некоторых
					данных. Обновление изображения
					производится каждый цикл -
					не рекомендуется использовать
					этот параметр при размерах лабиринта
					больших, чем размер терминала (~20*20).

  [-f ЗНАЧЕНИЕ|--fps ЗНАЧЕНИЕ]		Позволяет замедлить игру
					(внутренние расчёты) или попытаться
					ускорить (при запуске на достаточно
					быстродействующем оборудовании,
					если данная настройка ранее была
					синхронизирована с обновлением экрана -
					возможно снятие этого ограничения).

  [-F|--fullscreen]			Указывает на необходимость запуска игры
					в полноэкранном режиме.

  [-H ЗНАЧЕНИЕ|--screen-height ЗНАЧЕНИЕ]	Устанавливает высоту окна
					(разрешение) игры.
					В оконном режиме ЗНАЧЕНИЕ
					может быть любым положительным
					числом, не более того, которое
					поддерживается видеоадаптером.
					В полноэкранном режиме ЗНАЧЕНИЕ
					может быть только одним из значений,
					которые поддерживаются
					видеоадаптером (если будет указано
					неподдерживаемое значение - будет
					произведена попытка найти наиболее
					близкое поддерживаемое разрешение,
					или установлено значение по умолчанию
					(разрешение рабочего стола).

  [-l ЗНАЧЕНИЕ|--length ЗНАЧЕНИЕ]	Устанавливает максимальную длину
					лабиринта. ЗНАЧЕНИЕ может быть целым
					числом в диапазоне от 5 до 1000
					включительно (по умолчанию - 11).

  [-n|--no-wall-removing]		Отключает возможность создания
  					цикличного лабиринта (не убирает
  					случайные стены в лабиринте).

  [-N [ЗНАЧЕНИЕ] |--new-conf [ЗНАЧЕНИЕ] ]	Позволяет использовать файл
					настройки, отличный от файла
					по умолчанию. Если указанный
					в ЗНАЧЕНИЕ файл не существует -
					он будет создан заного и заполнен
					значениями по умолчанию.
					Если у пользователя нет прав
					на чтение и запись в указанный
					каталог - будет использован
					файл по умолчанию
					(без перезаписи последнего).
					Если ЗНАЧЕНИЕ (файл) не указано -
					будет пересоздан и использован
					файл по умолчанию.

  [-p ЗНАЧЕНИЕ|--pits ЗНАЧЕНИЕ]		Параметр указывает на количество
					дыр (ям) в полу, через которые
					участники могут практически мгновенно
					(контролируется параметром
					'-T|--pit-time') перемещаться
					между случайными точками
					лабиринта (координаты вычисляются
					при генерации лабиринта).
					ЗНАЧЕНИЕ может быть равно "0"
					(без кавычек) - полное отсутствие
					дыр, или быть целым
					положительным числом больше 1.

  [-r ЗНАЧЕНИЕ|--rivals ЗНАЧЕНИЕ]	Устанавливает количество соперников
					(включая игрока), участвующих в игре.
					ЗНАЧЕНИЕ должно быть больше 0.

  [-R|--result]				Отображает окончательный вариант
					вычисленного скелета лабиринта.

  [-s ЗНАЧЕНИЕ|--speed ЗНАЧЕНИЕ]	Устанавливает скорость отзывчивости
					игры на обработку ввода
					от пользователя при перемещении
					между соседними клетками лабиринта
					(фактически, регулирует скорость
					перемещения между ними).
					ЗНАЧЕНИЕ может быть положительным
					целым числом, чем меньше -
					тем меньше будет задержка
					обработки ввода (перемещение ускорится).

  [-S ЗНАЧЕНИЕ|--speed-turn ЗНАЧЕНИЕ]	Устанавливает скорость отзывчивости
					игры на обработку ввода
					от пользователя при повороте
					(фактически, регулирует
					скорость поворота).
					ЗНАЧЕНИЕ может быть положительным
					целым числом, чем меньше -
					тем меньше будет задержка
					обработки ввода (поворот ускорится).

  [-t ЗНАЧЕНИЕ|--trap-time ЗНАЧЕНИЕ]	Устанавливает время, которое
					участники теряют при
					попадании в ловушку.
					ЗНАЧЕНИЕ указывается в секундах,
					допустимый диапазон - от 0 до 30.

  [-T ЗНАЧЕНИЕ|--pit-time ЗНАЧЕНИЕ]	Устанавливает время, которое
					участники тратят на перемещение
					между дырами (ямами).
					ЗНАЧЕНИЕ указывается в секундах,
					допустимый диапазон - от 0 до 30.

  [-v ЗНАЧЕНИЕ|--vsync ЗНАЧЕНИЕ]	Включает или отключает использование
					синхронизации вертикального
					обновления экрана.
					ЗНАЧЕНИЕ вида "1", "y", "Y" или "on"
					(без кавычек) включает данную
					возможность, ЗНАЧЕНИЕ вида "0", "n", "N"
					или "off" (без кавычек) - выключает.
					Данный параметр конфликтует
					с параметром '--sdl12' и отключает его
					(если указан позднее).

  [-w ЗНАЧЕНИЕ|--width ЗНАЧЕНИЕ]	Устанавливает максимальную ширину
  					лабиринта. ЗНАЧЕНИЕ может быть целым
  					числом в диапазоне от 5 до 1000
  					включительно (по умолчанию - 11).

  [-W ЗНАЧЕНИЕ|--screen-width ЗНАЧЕНИЕ]	Устанавливает ширину окна
					(разрешение) игры.
					В оконном режиме ЗНАЧЕНИЕ
					может быть любым положительным
					числом, не более того, которое
					поддерживается видеоадаптером.
					В полноэкранном режиме ЗНАЧЕНИЕ
					может быть только одним из значений,
					которые поддерживаются
					видеоадаптером (если будет указано
					неподдерживаемое значение - будет
					произведена попытка найти наиболее
					близкое поддерживаемое разрешение,
					или установлено значение по умолчанию
					(разрешение рабочего стола).

  [--sdl12]				Параметр указывает на необходимость
					использования программного ускорения
					с помощью старого API SDL 1.2
					(может лучше работать на некоторых
					старых машинах/системах).
					Этот параметр конфликтует с параметрами
					'-a|--hw_accel' и '-v|--vsync'
					и отключает их (если указан позднее).

  [-h]					Показать краткую справку
					по использованию и выйти.

  [--help]				Показать полную справку с описанием
					возможных параметров и выйти.
